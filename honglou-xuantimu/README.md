# 《紅樓夢》論文選題思維導圖

互動式思維導圖，幫助學生理解論文選題的正確流程：從文本細讀出發，避免理念先行。

## 功能特點

- **雙模式對比**：展示正確方法（從文本出發）和錯誤示範（理念先行）的對比
- **互動式節點**：點擊每個步驟查看詳細說明
- **回溯機制視覺化**：展示如何可以回到前面的步驟進行迭代完善
- **框架優化對比**：展示如何用獨特思考優化常見框架

## 使用說明

1. 打開應用後，可以在頂部切換「正確方法」和「錯誤示範」兩種模式
2. 點擊思維導圖中的任意節點，查看該步驟的詳細說明
3. 在正確方法模式下，可以看到回溯箭頭，表示可以回到前面的步驟進行迭代

## 教學目標

幫助學生理解：
- 論文選題應該從文本細讀出發，而不是理念先行
- 關鍵詞要具象、可操作，能直接找到 primary sources
- Primary sources 必須是可摘錄的原文證據
- 可以使用常見框架，但要用獨特思考去優化
- 研究是一個迭代過程，可以回溯完善

## 適用對象

9 年級以上學生
