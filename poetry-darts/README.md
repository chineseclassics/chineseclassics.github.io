# 詩詞飛鏢 (Poetry Darts 3D)

太虛幻境中的 3D 手勢互動教育遊戲，將古典文學記憶與物理競技樂趣完美結合。

## 🌟 遊戲特點

- **手勢驅動**：利用 MediaPipe 技術實現專業級手勢控靶，無需鼠標鍵盤。
- **雙重模式**：
  - **填空模式**：在靶盤上尋找詩句中的缺失字並精準投擲。
  - **意象模式**：射中詩句對應的三個關鍵意象 (Emoji)，深入理解詩歌意境。
- **沉浸式體驗**：
  - 基於 Three.js 的 3D 競技場。
  - 直覺化的「磁力吸附」抓取感。
  - Web Audio API 實現的動態音效反饋。
- **寓教於樂**：從《意moji》數據庫精選 300+ 經典詩句，在投擲中加深記憶。

## 🎮 操作指南

1. ✋ **喚醒**：面向攝像頭張開手掌，等待能量環填滿。
2. 🎯 **瞄准**：移動手掌控制準心位置。
3. 👌 **抓取**：準心靠近飛鏢時，捏合手指（拇指與食指）進行鎖定。
4. 🚀 **投擲**：保持捏合並快速揮動手臂，在釋放點鬆開手指。

## 🛠️ 技術棧

- **Frontend**: Vue 3 (Composition API)
- **3D Engine**: Three.js
- **AI/Vision**: MediaPipe Hands
- **UI/Styles**: Tailwind CSS, LXGW WenKai TC
- **Audio**: Web Audio API

## 📁 目錄結構

- `js/core/`: 核心引擎（3D 場景、手勢管理、音效）
- `js/data/`: 詩詞數據庫
- `assets/`: 靜態資源
