# 紅樓舊夢 v2 - 遊戲流程模擬

## 一局完整遊戲流程模擬

### 遊戲開始

**初始狀態**：
- 玩家作為石頭（寶玉），站在一片白地的大觀園中
- 初始資源：絳珠 0 顆，靈石 0 顆
- 行動力：0/4（尚未開始第一個節氣）
- 已讀回目：0/80
- 已解鎖記憶：0
- 已解鎖建築：0
- 已種植花魂：0

---

## 第一階段：記憶解鎖與資源獲取（第 1-3 回）

### 節氣 1：立春（遊戲開始）

**開始節氣**：
- 獲得 4 點行動力
- 節氣顯示：「立春」
- 記憶列表顯示：2 個未解鎖記憶
  - 「第 1 回的記憶」（灰色，未解鎖）
  - 「第 3 回的記憶」（灰色，未解鎖）

**玩家行動**：
1. **點擊「第 1 回的記憶」**（不消耗行動力）
   - 系統顯示選擇題：「這回中，石頭最初在哪裡？」（標註「第 1 回的記憶」）
   - A. 青埂峰  B. 大荒山  C. 無稽崖  D. 太虛幻境
   - 玩家第一次嘗試，選擇 B（錯誤）
   - 提示：「建議查看第 1 回原文」
   - 玩家離開遊戲，閱讀第 1 回「甄士隱夢幻識通靈 賈雨村風塵懷閨秀」
   - 回到遊戲，再次答題，選擇 A（正確）
   - **記憶解鎖：「石頭記的起源」（stone 類記憶）**
   - **直接獲得靈石：20 顆**
   - 顯示獲得資源動畫

2. **點擊「第 3 回的記憶」**（不消耗行動力）
   - 系統顯示選擇題：「這回中，黛玉初入榮國府時，第一個見到的主要人物是誰？」（標註「第 3 回的記憶」）
   - A. 賈寶玉  B. 王夫人  C. 賈母  D. 賈政
   - 玩家第一次嘗試，選擇 A（錯誤）
   - 提示：「建議查看第 3 回原文」
   - 玩家離開遊戲，閱讀第 3 回「賈雨村夤緣復舊職 林黛玉拋父進京都」
   - 回到遊戲，再次答題，選擇 C（正確）
   - **記憶解鎖：「初入榮府」（pearl 類記憶）**
   - **直接獲得絳珠：10 顆**
   - 顯示獲得資源動畫

3. **解鎖建築**（消耗靈石：20 顆，不消耗行動力）
   - 選擇解鎖「瀟湘館」
   - 建築解鎖成功
   - 提示：「瀟湘館已解鎖，可以在這裡種植黛玉花魂」

4. **種植花魂**（消耗 2 點行動力）
   - 在瀟湘館中種植「黛玉芙蓉魂」
   - 花魂種植成功，等級：0 級，成長值：0/100

**當前狀態**：
- 行動力：2/4
- 絳珠：10 顆，靈石：0 顆
- 已解鎖記憶：2 個（「石頭記的起源」、「初入榮府」）
- 已解鎖建築：1 個（瀟湘館）
- 已種植花魂：1 個（黛玉芙蓉魂，Lv0）

**進入下一節氣**

## 第二階段：持續答題解鎖與培養（第 5-27 回）

### 節氣 4-10：持續循環

**每個節氣的完整循環**：
1. **答題解鎖記憶**（不消耗行動力）
   - 點擊記憶，回答選擇題（標註「第 X 回的記憶」）
   - 如果答錯，玩家自然會去閱讀第 X 回
   - 答對後，記憶解鎖 + 直接獲得資源（絳珠或靈石）

2. **使用資源進行行動**（消耗行動力）
   - 使用靈石解鎖建築（不消耗行動力）
   - 使用靈石解鎖建築後，種植花魂（消耗 2 點行動力）
   - 使用絳珠澆灌花魂（消耗 1 點行動力）

3. **花魂升級，解鎖新記憶**
   - 花魂達到特定等級後，解鎖新記憶
   - 回到步驟 1

## 流程總結

### 核心循環（每個節氣）

**每個節氣都包含完整的行動循環**：

1. **答題解鎖記憶**（不消耗行動力）
   - 記憶列表顯示「第 X 回的記憶」（灰色，未解鎖）
   - 點擊記憶，回答選擇題（標註「第 X 回的記憶」）
   - 如果答錯，玩家自然會去閱讀第 X 回（遊戲外）
   - 答對後，記憶解鎖 + 直接獲得資源（絳珠或靈石）

2. **使用資源進行行動**（消耗行動力）
   - **解鎖建築**（消耗靈石，不消耗行動力）
   - **種植花魂**（消耗 2 點行動力）
   - **澆灌花魂**（消耗 1 點行動力，消耗絳珠）

3. **花魂升級，解鎖新記憶**
   - 花魂等級提升
   - 解鎖新記憶
   - 回到步驟 1

**重要原則**：
- 每個節氣都應該包含「答題解鎖記憶」和「使用資源進行行動」兩個部分
- 答題解鎖記憶不消耗行動力，可以隨時進行
- 使用資源進行行動消耗行動力，需要策略性分配

### 關鍵設計點

1. **閱讀系統隱性化**：不在遊戲中明確告訴玩家「去閱讀」，而是通過題目自然引導
2. **答對即解鎖**：答對選擇題後，記憶立即解鎖並獲得資源，不需要額外的「搜尋記憶」步驟
3. **題目提示回目**：題目會顯示「第 X 回的記憶」，玩家為了答對題目，自然會去閱讀
4. **資源平衡**：pearl 類記憶（較少，絳珠較多）和 stone 類記憶（較多，靈石較少）的平衡
5. **循環推進**：發現記憶 → 答題解鎖 → 獲得資源 → 解鎖建築 → 種植花魂 → 澆灌培養 → 解鎖新記憶
6. **漸進式引導**：遊戲進度引導閱讀順序，閱讀進度推動遊戲進度
