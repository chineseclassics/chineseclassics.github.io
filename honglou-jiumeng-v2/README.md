# 紅樓舊夢 v2 - 大觀園記憶重建遊戲

> 玩家作為石頭（寶玉），用林黛玉的絳珠淚灌溉大觀園花魂，重建園中景致與眾人的美好記憶。

## 🎮 遊戲目標

重建完整的大觀園，復活所有花魂，收集所有大觀園美好記憶，最終完成一部屬於自己的《石頭記》。

## 📋 MVP 功能

- ✅ 節氣循環系統（24 節氣）
- ✅ 行動力系統（每節氣 4 點）
- ✅ 資源系統（絳珠淚、靈石）
- ✅ 園林地圖（5×5 格子）
- ✅ 建築解鎖（瀟湘館）
- ✅ 花魂系統（黛玉花魂，等級 0-5）
- ✅ 絳珠淚系統（2-3 種淚水類型）
- ✅ 記憶系統（1 條黛玉故事線）
- ✅ 簡單終局總結

## 🚀 快速開始

1. 直接在瀏覽器中打開 `index.html`
2. 或使用本地服務器：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (需要安裝 http-server)
   npx http-server
   ```
3. 訪問 `http://localhost:8000`

## 🎯 遊戲玩法

1. **收集資源**：透過搜尋記憶獲得絳珠淚和靈石
2. **解鎖建築**：使用靈石解鎖瀟湘館
3. **種植花魂**：在瀟湘館種植黛玉花魂
4. **澆灌成長**：使用絳珠淚澆灌花魂，提升等級
5. **收集記憶**：搜尋並解鎖記憶故事線
6. **達成終局**：當黛玉花魂達到 5 級時觸發終局
