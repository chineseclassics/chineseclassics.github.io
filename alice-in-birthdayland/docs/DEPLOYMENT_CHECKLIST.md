## 部署步驟

### 1. 本地測試
```bash
# 在本地啟動服務器測試
cd alice-in-birthdayland
python -m http.server 8000
# 或使用 Node.js
npx serve .
```

### 2. 文件上傳
```bash
# 確保所有文件都在正確位置
# 檢查文件權限
chmod -R 755 alice-in-birthdayland/
```

### 3. 太虛幻境整合
