# 忘川・詩魂渡 開發計劃

## 1. 項目概述

**忘川・詩魂渡** 是一款基於詩詞意象解謎的教育遊戲。玩家扮演孟婆的學徒，幫助喝了孟婆湯但仍有執念殘留的詩人靈魂。通過探索詩人的「殘夢圖」，提取視覺意象（墨影），煉化為詩句，最終配製詩魂湯，清空靈魂重量，幫助詩人登船渡河。

## 2. 核心概念與機制

### 2.1 世界觀設定

**地點**：忘川河畔的「停雲渡」碼頭

**核心角色**：

- **玩家**：孟婆的學徒，正在學習如何處理特殊靈魂
- **NPC**：孟婆（師父）、詩人靈魂（首發：李商隱）、船夫

**核心矛盾**：

- **孟婆湯的困境**：孟婆煎製的普通版孟婆湯對詩人無效（只清空了70%，仍殘留30%執念）
- **執念的特殊性**：詩人的執念不是「事」（具體的人和經歷），而是「感」（極致的感官體驗）
- **靈魂超重**：殘留執念導致靈魂過重，登船後船身下沉，無法渡河
- **學徒的任務**：孟婆把這個疑難案例交給你處理，作為考核
- **經典引用**：「只恐雙溪舴艋舟，載不動許多愁」——詩人的愁緒太重，船載不動
- **昇華原理**：詩人的執念只能用詩人自己的詩來化解——當年他把沉重的感受昇華為詩句時，執念已經找到了出口。現在把這些詩句煎成湯，執念會再次經歷昇華過程，自然消散。不是讓他「記起」，而是讓執念「重走當年的昇華之路」後消失

### 2.2 核心邏輯：殘夢圖與詩句煉化

**你的發現**：

- 詩人的靈魂深處有一幅「殘夢圖」，包含了所有喝了孟婆湯也忘不掉的執念意象
- 當你閉眼感知詩人靈魂時，不僅能看到殘夢圖，還能感受到那些意象背後的記憶碎片
- 你需要從殘夢圖中提取意象（墨影），配對煉化為詩句
- 詩句凝結成詩箋，收集到詩箋匣中
- 收集齊所有詩箋後，投入湯中配製詩魂湯

**邏輯鏈**：

```
殘夢圖（忘不掉的執念圖像）→ 點擊區域 → 閱讀執念描述 → 提取墨影 → 配對煉化 → 生成詩箋 → 收集所有詩箋 → 投入湯中 → 配成詩魂湯 → 詩人服用 → 清空
```

**為什麼邏輯自洽**：

- 視覺隱喻：混亂的圖像（執念）→ 純淨的文字（詩句）= 昇華
- 殘夢圖保持不變，只作為探索與提取的場景
- 詩箋收集完成後，投入湯中配製詩魂湯
- 你能從殘夢中感知記憶碎片，用於記錄見聞錄

### 2.3 軟挑戰性機制：靈力系統

**設計目的**：增加思考深度，避免無腦試錯，但不影響通關

**靈力設定**：

- 初始靈力：100點
- 每次錯誤配對消耗：20點靈力
- 靈力歸零時：需要等待30秒自動恢復到滿（或點擊「喝茶休息」立即恢復）
- 查看《孟婆秘方》提示：不消耗靈力
- 正確煉化：不消耗靈力

**視覺反饋**：

```
┌─────────────────────────┐
│ 靈力：████████░░ 80/100 │
└─────────────────────────┘
```

**孟婆的解釋**（首次煉化前）：

```
孟婆：「煉化詩句需要消耗你的靈力。
如果配對錯誤，靈力會流失一些。
不過別擔心，休息一下就能恢復。
如果拿不準，可以先看看我給你的秘方。」
```

**平衡性**：

- 100點靈力 ÷ 20點/次 = 最多錯5次
- 5首詩，平均每首可以錯1次
- 不會卡住學生，只是鼓勵思考

### 2.4 遊戲循環

```
 詩人來到孟婆茶攤 → 喝下普通孟婆湯
    ↓
去渡口登船 → 船身下沉！（超重）
    ↓
孟婆對你（學徒）：「這個案子交給你處理」
    ↓
你閉眼感知詩人靈魂 → 展開「殘夢圖」（AI生成圖片，包含10個意象）
    ↓
【探索階段】玩家在殘夢圖上點擊任意區域
    ↓
  點擊區域 → 彈出該區域的「執念描述」（感官日記文字）
    ↓
  閱讀描述 → 理解這是什麼意象
    ↓
  拖動該區域到「煉詩爐」→ 該區域從殘夢圖剝離 → 變成墨影（如🌧️ 夜雨）
    ↓
  殘夢圖上該區域變暗（已提取標記）
    ↓
【煉化階段】當爐中有兩道墨影時
    ↓
  點擊「煉化」→ 判斷組合
    ↓
  ├─ 正確 → 詩句浮現 → 凝結成詩箋
  │     ↓
  │  獲得詩箋（收入詩箋匣，同時記錄到《忘川見聞錄》）
  │     ↓
  │  煉詩爐清空，可繼續提取新墨影
  │
  └─ 錯誤 → 消耗20點靈力 → 墨影返回殘夢圖 → 重新選擇
      ↓
      如果靈力歸零 → 需要休息30秒（或喝茶立即恢復）
    ↓
重複探索與煉化，直到收集齊所有詩箋（5首詩）
    ↓
【配湯階段】詩箋收集完成（5/5）
    ↓
將所有詩箋投入大湯罐 → 詩句墨跡融化 → 湯變金色
    ↓
詩人服用詩魂湯 → 所有詩句閃回 → 靈魂徹底清空
    ↓
登船 → 船身浮起 → 渡河成功
```

## 3. 核心術語體系

| 概念               | 術語           | 說明                                                                    |
| ------------------ | -------------- | ----------------------------------------------------------------------- |
| 詩人靈魂的視覺化   | 殘夢圖         | AI生成的圖片，包含12個意象元素（喝了孟婆湯也忘不掉的執念）              |
| 圖片上的可點擊區域 | 執念區域       | 對應不同意象的視覺元素                                                  |
| 點擊後顯示的文字   | 執念描述       | 感官體驗 + 記憶碎片（如「雨聲淅瀝...被貶入蜀，長夜聽雨...」）           |
| 提取後的具體意象   | 墨影           | 從殘夢圖提取的意象（如「夜雨墨影」🌧️）                                |
| 存放墨影的地方     | 煉詩爐         | 核心互動區域                                                            |
| 已提取待選區       | 墨影托盤       | 顯示已提取但未煉化的墨影，可放入煉詩爐或放回殘夢圖                      |
| 煉化產物           | 詩箋           | 帶有詩句文字的紙片                                                      |
| 最終產物           | 詩魂湯         | 融入所有詩句的特製湯藥                                                  |
| 學徒的秘密任務     | 《忘川見聞錄》 | 記錄詩人生平的小冊子（信息來自殘夢中感知的記憶碎片）                    |
| 意象收集系統       | 《意象圖鑑》   | 學徒筆記的一部分，記錄不同詩人對同一意象的不同處理方式                  |
| 干擾意象           | 無法配對的墨影 | 12個意象中有2個無法配對成功（增加思考深度，避免無腦試錯）               |
| 煉化失敗懲罰       | 靈力消耗       | 錯誤配對消耗20點靈力，歸零需休息恢復（軟挑戰，不會卡住玩家）            |

**術語邏輯**：

```
殘夢圖（忘不掉的執念圖像）→ 點擊執念區域 → 閱讀執念描述 → 提取墨影 → 配對煉化 → 生成詩句（詩箋）→ 收集所有詩箋 → 投入湯中 → 配成詩魂湯
```

## 5. 詳細遊戲流程

### 階段一：開場與任務交接

**場景**：忘川河畔，停雲渡碼頭

```
【畫面】
迷霧繚繞的河面上，停泊著一艘孤舟。
碼頭旁，孟婆坐在茶攤前，正在煎湯。
你（學徒）在旁邊幫忙燒火。

一個模糊的詩人身影走來。

孟婆：「來，喝下這個。」

詩人接過碗，一飲而盡。

孟婆：「去登船吧。」

詩人走向渡船，剛踏上船板——

【船身劇烈下沉，幾乎沒入水中】

船夫（搖頭）：「孟婆，這位太重了，載不動。」

孟婆（嘆氣）：「湯沒喝透...又是詩人。」

孟婆轉頭對你說：
「小徒弟，這種案子正好給你練手。
他的執念太深，普通湯洗不掉，
你去幫他配一碗特製的。」

你（緊張）：「我...我行嗎？」

孟婆（微笑）：「我教過你的。
詩人的執念，得用詩人自己的詩來化解。
當年他把這些沉重的感受昇華為詩句時，
執念就找到了出口、釋放掉了。
現在把這些詩句煎成湯，執念會再次經歷昇華，
自然消散，不留痕跡。」

你：「可是...他現在應該什麼都不記得了吧？」

孟婆：「沒錯，他連自己是誰都忘了。
但執念還在，像夢魘一樣糾纏著他的靈魂。」

你：「那詩魂湯是要讓他想起來嗎？」

孟婆（搖頭）：「不，不是讓他『想起』，而是讓執念『消散』。
當年他把這些感受寫成詩時，就是一次昇華——
把沉重的執念化為輕盈的文字，釋放掉了。
現在把這些詩句煎成湯，就是讓執念重新經歷那次昇華，
像當年一樣自然消散，不留痕跡。」

你：「所以他不會記得這些詩？」

孟婆：「對，他什麼都不會記得。
詩句只是引子，幫執念找到當年的出口。
執念消散後，詩句也會隨之散去，
留下的只是輕盈的靈魂。」

孟婆（拍拍你的肩）：
「他的執念都在殘夢裡，你進去看看就知道了。」
```

### 階段二：展開殘夢圖

**場景**：詩人在茶攤前坐下

```
你（學徒）走到詩人面前，深吸一口氣。

你：「讓我看看您的靈魂...」

你閉上眼，將手放在詩人頭上。

【畫面切換】
一幅巨大的「殘夢圖」緩緩展開。

這是一幅夢幻般的水墨畫，包含詩人喝了孟婆湯也忘不掉的執念：
- 右上角：雨絲與山巒交織
- 左側：蠟燭與春蠶
- 中央：蝴蝶在霧中飛舞
- 下方：明珠在月光下閃爍
... 10個意象的視覺元素分散在畫面各處

孟婆（旁白）：「看到了嗎？這幅『殘夢圖』就是他揮之不去的執念。
把這些意象提取出來，煉化成詩句，他就能放下了。」
```

**UI界面**：

```

│  靈力：████████░░ 80/100            │
├─────────────────────────────────────┤
│                                      │

│                                      │

│        探索其中的執念                │
│                                      │
├─────────────────────────────────────┤
│  煉詩爐：                            │

│                                      │

│  [查看秘方]（不消耗靈力）            │

```

**操作流程**：

**1. 點擊殘夢圖上的某個區域（如：雨絲）**

```
【彈出模態框】
┌─────────────────────────────────────┐
│  執念殘影：夜雨                      │
├─────────────────────────────────────┤
│  執念描述：                          │
│  「雨聲淅瀝，一直下個不停...        │
│   心裡濕漉漉的，壓得喘不過氣。      │

│   我孤獨的心。」                    │
│                                      │
│  ───────────────────────────        │
│  記憶碎片（你感知到的）：            │
│  「被貶入蜀...長夜聽雨...」          │
│                                      │
│  [提取這道墨影] [關閉]              │
└─────────────────────────────────────┘
```

**2. 點擊「提取這段執念」**

```
【動畫】
  該區域發光，從殘夢圖上「剝離」，
  飛向煉詩爐，變成一道發光的墨影。
  
  【煉詩爐】
  顯示：🌧️ 夜雨
  
  【殘夢圖】
  該區域變暗並標記「已提取」（灰色遮罩）
```

**3. 繼續點擊其他區域，收集第二道墨影**

```
玩家點擊「山巒」區域
    ↓
彈出執念描述：「山影重重...遙不可及...」
    ↓
提取 → 獲得🏔️ 巴山
    ↓
煉詩爐：[🌧️ 夜雨] + [🏔️ 巴山]
```

**4. 點擊「煉化」**

```
【判斷組合】

正確組合：
  火焰金光大作
      ↓
```

## 9. 視覺演出重點

### 關鍵動畫場景

**1. 意象提取**

- 點擊區域 → 區域發光 → 剝離飛向煉詩爐 → 變成墨影
- 殘夢圖上該區域變暗並顯示灰色遮罩（已提取標記）

**2. 詩句煉化**

- 煉化成功 → 詩句浮現在爐中 → 凝結成詩箋 → 收入詩箋匣
- 殘夢圖上對應區域完全消失或變成金色標記（已煉化）
- 煉化失敗 → 墨影返回殘夢圖 → 灰色遮罩消失，恢復可點擊

**3. 詩箋投入湯中**

- 所有詩箋飛入湯罐 → 墨跡溶解 → 湯變金色

**4. 服藥與清空**

- 詩人喝湯 → 所有詩句環繞身體 → 閃回記憶 → 詩句化光消散 → 詩人變清透

## 10. 技術實現要點

### 殘夢圖互動

**方案A：HTML Image Map**

- 優點：實現簡單
- 缺點：熱區是固定多邊形，不夠靈活

**方案B：Canvas + 座標檢測**（推薦）

- 優點：可以動態添加/移除熱區、實現動畫、標記已提取區域
- 缺點：需要手動標註座標

**方案C：SVG + 圖層**

- 優點：每個意象是獨立圖層，可單獨控制
- 缺點：需要設計師配合

### 解鎖與成就系統

**單個詩人通關獎勵**：
- ✅ 解鎖「詩人專題頁」（可重溫該詩人所有詩作和生平）
- ✅ 解鎖「詩人圖鑑」中的該詩人條目
- ✅ 獲得「孟婆學徒徽章・初階」

**多個詩人完成獎勵**：
- 完成3位詩人 → 解鎖「忘川渡口全景圖」（可自由探索）
- 完成5位詩人 → 解鎖「特殊詩人」（如蘇軾、李清照）
- 完成10位詩人 → 升級為「正式孟婆」，解鎖特殊劇情

**收藏系統**：
- 詩人圖鑑：記錄所有已渡河的詩人
- 詩作收藏：可隨時重溫所有煉化過的詩句
- 見聞錄檔案：保存所有詩人的生平記錄

## 14. 核心機制設計理念

### 故事邏輯自洽性

**1. 詩人的失憶狀態與昇華機制**
- 喝了普通孟婆湯後：連自己是誰都忘了（完全失憶）
- 執念殘留：但感官體驗（意象）像夢魘般揮之不去
- 詩魂湯的作用：不是讓他「想起」，而是讓執念「再次昇華」
  - 當年寫詩 = 執念昇華為文字（第一次釋放）
  - 現在喝湯 = 執念再走昇華之路（徹底消散）
  - 詩句是引子，幫執念找到當年的出口
  - 過程結束後，詩人不會記得任何東西，只是變輕盈了

**2. 見聞錄信息來源**
- 學徒的特殊能力：探索殘夢時能感知記憶碎片
- 記憶碎片來自：每個意象背後的模糊記憶
- 拼湊過程：煉化兩個意象時，兩個碎片互相印證，形成完整記錄

**3. 靈力系統的故事化**
- 不是遊戲化的血條，而是學徒的精神消耗
- 錯誤配對消耗心神，需要休息恢復
- 查看秘方是智慧而非蠻力，不消耗靈力
- 孟婆鼓勵你：「拿不準就看秘方，別硬來」

### 軟挑戰性設計哲學

**目標**：增加思考深度，但不製造挫折感

**實現手段**：
1. **12配5意象設計**：提供干擾項，避免無腦試錯
2. **靈力消耗**：錯誤有代價，但可以恢復
3. **秘方系統**：卡住時隨時可查提示
4. **無死局**：任何情況都能通關，只是時間問題

**平衡性測試**：
- 對詩歌完全陌生的學生：可以全程看秘方，仍能學到知識
- 對詩歌有一定了解的學生：挑戰自己記憶，獲得成就感
- 資深詩詞愛好者：追求零失誤通關，獲得滿足感

### 教育與遊戲的平衡

**教育優先**：
- 不因難度卡住學生
- 多層級提示系統
- 通關後豐富的詩作與生平展示

**遊戲化增強**：
- 靈力系統增加儀式感
- 干擾項增加策略性
- 見聞錄增加收集樂趣

## 16. 意象圖鑑系統總結

### 核心價值

**對學生的教育價值**：
1. **單詩層面**：理解意象在詩中的作用（情感的寄託）
2. **跨詩層面**：發現相同意象在不同詩人筆下的差異
3. **詩史層面**：理解意象在中國詩歌史中的演變和互文傳統
4. **文化層面**：了解意象背後的文化背景和象徵意義

**對遊戲的增強**：
1. **收集動機**：除了渡河主線，增加長期收集目標
2. **重玩價值**：為了收集完整意象，會回訪不同詩人
3. **成就系統**：解鎖專題、獲得徽章，增加成就感
4. **知識深度**：從背誦詩句到理解詩學，層次提升

### 實施要點

**簡潔優先**：
- 自動記錄，無需學生手動操作
- 不影響主線流程
- 可選查看，不強制閱讀

**漸進揭示**：
- 第1次：基礎認識（這是什麼意象）
- 第2次：對比發現（互文性）
- 第3次：專題解鎖（文化深度）

**視覺美感**：
- 手寫筆記風格
- 翻頁動畫
- 古典書卷美學

### 與主線的關係

```
主線任務：
渡河詩人（必須完成）
    ↓
獲得詩箋、見聞錄、意象記錄
    ↓
解鎖下一位詩人

支線收集：
意象圖鑑（可選深入）
    ↓
收集更多版本
    ↓
解鎖專題和成就
    ↓
提升詩詞鑑賞能力
```

**平衡設計**：
- 主線不依賴圖鑑（即使不看圖鑑也能通關）
- 圖鑑提供額外獎勵（鼓勵深入學習）
- 成就系統滿足收集欲（提升重玩性）
