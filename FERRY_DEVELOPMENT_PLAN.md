# 忘川・詩魂渡 開發計劃

## 1. 項目概述

**忘川・詩魂渡** 是一款基於詩詞意象解謎的教育遊戲。玩家扮演孟婆的學徒，幫助喝了孟婆湯但仍有執念殘留的詩人靈魂。通過探索詩人的「殘夢圖」，提取視覺意象（墨影），煉化為詩句，最終配製詩魂湯，清空靈魂重量，幫助詩人登船渡河。

## 2. 核心概念與機制

### 2.1 世界觀設定

**地點**：忘川河畔的「停雲渡」碼頭

**核心角色**：

- **玩家**：孟婆的學徒，正在學習如何處理特殊靈魂
- **NPC**：孟婆（師父）、詩人靈魂（首發：李商隱）、船夫

**核心矛盾**：

- **孟婆湯的困境**：孟婆煎製的普通版孟婆湯對詩人無效（只清空了70%，仍殘留30%執念）
- **執念的特殊性**：詩人的執念不是「事」（具體的人和經歷），而是「感」（極致的感官體驗）
- **靈魂超重**：殘留執念導致靈魂過重，登船後船身下沉，無法渡河
- **學徒的任務**：孟婆把這個疑難案例交給你處理，作為考核
- **經典引用**：「只恐雙溪舴艋舟，載不動許多愁」——詩人的愁緒太重，船載不動

### 2.2 核心邏輯：殘夢圖與詩句煉化

**你的發現**：

- 詩人的靈魂深處有一幅「殘夢圖」，包含了所有喝了孟婆湯也忘不掉的執念意象
- 你需要從殘夢圖中提取意象（墨影），配對煉化為詩句
- 詩句凝結成詩箋，收集到詩箋匣中
- 收集齊所有詩箋後，投入湯中配製詩魂湯

**邏輯鏈**：

```
殘夢圖（忘不掉的執念圖像）→ 點擊區域 → 閱讀執念描述 → 提取墨影 → 配對煉化 → 生成詩箋 → 收集所有詩箋 → 投入湯中 → 配成詩魂湯 → 詩人服用 → 清空
```

**為什麼邏輯自洽**：

- 視覺隱喻：混亂的圖像（執念）→ 純淨的文字（詩句）= 昇華
- 殘夢圖保持不變，只作為探索與提取的場景
- 詩箋收集完成後，投入湯中配製詩魂湯

### 2.3 遊戲循環

```
 詩人來到孟婆茶攤 → 喝下普通孟婆湯
    ↓
去渡口登船 → 船身下沉！（超重）
    ↓
孟婆對你（學徒）：「這個案子交給你處理」
    ↓
你閉眼感知詩人靈魂 → 展開「殘夢圖」（AI生成圖片，包含10個意象）
    ↓
【探索階段】玩家在殘夢圖上點擊任意區域
    ↓
  點擊區域 → 彈出該區域的「執念描述」（感官日記文字）
    ↓
  閱讀描述 → 理解這是什麼意象
    ↓
  拖動該區域到「煉詩爐」→ 該區域從殘夢圖剝離 → 變成墨影（如🌧️ 夜雨）
    ↓
  殘夢圖上該區域變暗（已提取標記）
    ↓
【煉化階段】當爐中有兩道墨影時
    ↓
  點擊「煉化」→ 判斷組合
    ↓
  ├─ 正確 → 詩句浮現 → 凝結成詩箋
  │     ↓
  │  獲得詩箋（收入詩箋匣，同時記錄到《忘川見聞錄》）
  │
  └─ 錯誤 → 墨影返回殘夢圖 → 重新選擇
    ↓
重複探索與煉化，直到收集齊所有詩箋（5首詩）
    ↓
【配湯階段】詩箋收集完成（5/5）
    ↓
將所有詩箋投入大湯罐 → 詩句墨跡融化 → 湯變金色
    ↓
詩人服用詩魂湯 → 所有詩句閃回 → 靈魂徹底清空
    ↓
登船 → 船身浮起 → 渡河成功
```

## 3. 核心術語體系

| 概念               | 術語           | 說明                                                       |
| ------------------ | -------------- | ---------------------------------------------------------- |
| 詩人靈魂的視覺化   | 殘夢圖         | AI生成的圖片，包含所有意象元素（喝了孟婆湯也忘不掉的執念） |
| 圖片上的可點擊區域 | 執念區域       | 對應不同意象的視覺元素                                     |
| 點擊後顯示的文字   | 執念描述       | 感官日記（如「雨聲淅瀝...」）                              |
| 提取後的具體意象   | 墨影           | 從殘夢圖提取的意象（如「夜雨墨影」🌧️）                   |
| 存放墨影的地方     | 煉詩爐         | 核心互動區域                                               |
| 煉化產物           | 詩箋           | 帶有詩句文字的紙片                                         |
| 最終產物           | 詩魂湯         | 融入所有詩句的特製湯藥                                     |
| 學徒的秘密任務     | 《忘川見聞錄》 | 記錄詩人生平的小冊子                                       |

**術語邏輯**：

```
殘夢圖（忘不掉的執念圖像）→ 點擊執念區域 → 閱讀執念描述 → 提取墨影 → 配對煉化 → 生成詩句（詩箋）→ 收集所有詩箋 → 投入湯中 → 配成詩魂湯
```

## 5. 詳細遊戲流程

### 階段一：開場與任務交接

**場景**：忘川河畔，停雲渡碼頭

```
【畫面】
迷霧繚繞的河面上，停泊著一艘孤舟。
碼頭旁，孟婆坐在茶攤前，正在煎湯。
你（學徒）在旁邊幫忙燒火。

一個模糊的詩人身影走來。

孟婆：「來，喝下這個。」

詩人接過碗，一飲而盡。

孟婆：「去登船吧。」

詩人走向渡船，剛踏上船板——

【船身劇烈下沉，幾乎沒入水中】

船夫（搖頭）：「孟婆，這位太重了，載不動。」

孟婆（嘆氣）：「湯沒喝透...又是詩人。」

孟婆轉頭對你說：
「小徒弟，這種案子正好給你練手。
他的執念太深，普通湯洗不掉，
你去幫他配一碗特製的。」

你（緊張）：「我...我行嗎？」

孟婆（微笑）：「我教過你的。
他的執念都在記憶裡，你進去看看就知道了。」
```

### 階段二：展開殘夢圖

**場景**：詩人在茶攤前坐下

```
你（學徒）走到詩人面前，深吸一口氣。

你：「讓我看看您的靈魂...」

你閉上眼，將手放在詩人頭上。

【畫面切換】
一幅巨大的「殘夢圖」緩緩展開。

這是一幅夢幻般的水墨畫，包含詩人喝了孟婆湯也忘不掉的執念：
- 右上角：雨絲與山巒交織
- 左側：蠟燭與春蠶
- 中央：蝴蝶在霧中飛舞
- 下方：明珠在月光下閃爍
... 10個意象的視覺元素分散在畫面各處

孟婆（旁白）：「看到了嗎？這幅『殘夢圖』就是他揮之不去的執念。
把這些意象提取出來，煉化成詩句，他就能放下了。」
```

### 階段三：探索與提取（核心玩法）

**UI界面**：

```
┌─────────────────────────────────────┐
│         殘夢圖（全屏）               │
├─────────────────────────────────────┤
│                                      │
│  [AI生成的完整圖片，包含10個意象]   │
│                                      │
│  提示：點擊圖上的任何區域，          │
│        探索其中的執念                │
│                                      │
├─────────────────────────────────────┤
│  煉詩爐：                            │
│  [空位1] + [空位2] = ？             │
│  [煉化]（需要兩道墨影）              │
├─────────────────────────────────────┤
│  已收集詩箋：0/5                     │
└─────────────────────────────────────┘
```

**操作流程**：

**1. 點擊殘夢圖上的某個區域（如：雨絲）**

```
【彈出模態框】
┌─────────────────────────────────────┐
│  執念殘影                            │
├─────────────────────────────────────┤
│                                      │
│  「雨聲淅瀝，一直下個不停...        │
│   心裡濕漉漉的，壓得喘不過氣。      │
│   那聲音敲打著池塘，也敲打著        │
│   我孤獨的心。」                    │
│                                      │
│  [提取這段執念] [關閉]              │
└─────────────────────────────────────┘
```

**2. 點擊「提取這段執念」**

```
【動畫】
  該區域發光，從殘夢圖上「剝離」，
  飛向煉詩爐，變成一道發光的墨影。
  
  【煉詩爐】
  顯示：🌧️ 夜雨
  
  【殘夢圖】
  該區域變暗（已提取標記）
```

**3. 繼續點擊其他區域，收集第二道墨影**

```
玩家點擊「山巒」區域
    ↓
彈出執念描述：「山影重重...遙不可及...」
    ↓
提取 → 獲得🏔️ 巴山
    ↓
煉詩爐：[🌧️ 夜雨] + [🏔️ 巴山]
```

**4. 點擊「煉化」**

```
【判斷組合】

正確組合：
  火焰金光大作
      ↓
  詩句浮現（金色書法字）：
  「君問歸期未有期，
   巴山夜雨漲秋池。
   何當共剪西窗燭，
   卻話巴山夜雨時。」
      ↓
  詩句逐漸凝聚成一張發光的「詩箋」，
  紙片緩緩落下，被你用竹夾接住。
      ↓
  你將詩箋收入「詩箋匣」中。
      ↓
  【UI顯示】
  已收集詩箋：1/5
  
錯誤組合：
  火焰熄滅
      ↓
  提示：「這兩道墨影無法相融」
      ↓
  墨影返回殘夢圖原位置
```

### 階段五：配製詩魂湯

```

### 階段六：服用與渡河

```

詩人（眼神從迷茫逐漸變得清明）：

詩人（深呼吸，露出釋然的笑容）：

## 9. 視覺演出重點

### 關鍵動畫場景

**1. 意象提取**

- 點擊區域 → 區域發光 → 剝離飛向煉詩爐 → 變成墨影
- 殘夢圖上該區域變暗（已提取標記）

**2. 詩句煉化**

- 煉化成功 → 詩句浮現在爐中 → 凝結成詩箋 → 收入詩箋匣

**3. 詩箋投入湯中**

- 所有詩箋飛入湯罐 → 墨跡溶解 → 湯變金色

**4. 服藥與清空**

- 詩人喝湯 → 所有詩句環繞身體 → 閃回記憶 → 詩句化光消散 → 詩人變清透

## 10. 技術實現要點

### 殘夢圖互動

**方案A：HTML Image Map**

- 優點：實現簡單
- 缺點：熱區是固定多邊形，不夠靈活

**方案B：Canvas + 座標檢測**（推薦）

- 優點：可以動態添加/移除熱區、實現動畫、標記已提取區域
- 缺點：需要手動標註座標

**方案C：SVG + 圖層**

- 優點：每個意象是獨立圖層，可單獨控制
- 缺點：需要設計師配合
