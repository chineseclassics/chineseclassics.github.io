# 聶隱娘傳奇：羈絆與超脫

互動式文字 RPG，讓學生透過聶隱娘的一生體驗古典文學、品格抉擇與文言詞彙學習。玩家可在「思想內閣」與「八德一智」兩套系統間平衡，解鎖 18 種結局與多項成就。

## 目錄結構

```
yinniang/
├── index.html        # 應用入口（引用 Tailwind CDN）
├── css/
│   └── style.css     # 客製化樣式
├── js/
│   └── main.js       # 遊戲邏輯與狀態管理
├── assets/
│   ├── audio/        # 預留音效資源
│   ├── data/         # 預留資料檔
│   ├── fonts/        # 預留字體檔
│   └── images/       # 預留圖片檔
└── docs/
    └── DESIGN.md     # 完整遊戲設計與教學目標
```

## 開發與部署注意事項

- 所有資源必須儲存在 `yinniang/` 內，維持應用自包含。
- `index.html` 已在 `</body>` 前引用平台共用的 `taixu-app-switcher`，保持導航一致性。
- 若需新增靜態資源，請放入對應的 `assets/` 子目錄並更新引用路徑（相對於應用根目錄）。
- 遊戲邏輯集中在 `js/main.js`，如需擴充章節或系統，請在該文件中擴寫並維持繁體中文註解。
- 詳細的世界觀、系統設計與課程融入方式記錄於 `docs/DESIGN.md`，修改規劃前請先閱讀。
