# 紅樓舊夢

《紅樓舊夢》是一款以《紅樓夢》為背景的互動養成遊戲。玩家化身神瑛侍者，透過收集靈石與絳珠、重建大觀園、喚醒花魂與鳥靈，體驗「還淚緣」的輪迴旅程。

## 遊戲特色
- **節氣輪迴**：24 節氣對應四季變化，影響花魂成長與特殊事件。
- **資源管理**：收集絳珠與靈石，建造建築、澆灌花魂、修復衰敗。
- **記憶系統**：觸發林黛玉、賈寶玉的重要記憶，獲得特殊淚水或靈石。
- **鳥靈覺醒**：花魂達成條件即可喚醒丫鬟化身的鳥靈，提供被動增益。

## 使用方式
1. 於本地開啟 `index.html` 或部署至 GitHub Pages。
2. 確保頁面底部已載入 `/assets/js/taixu-app-switcher.js` 以使用平台導航。
3. 所有應用內容皆使用繁體中文。
