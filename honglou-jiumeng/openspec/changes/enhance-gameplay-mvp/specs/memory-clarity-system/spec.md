## ADDED Requirements

### Requirement: 記憶清晰度（clarity）
每段記憶擁有 `clarity:0–100`；抽取/重溫時提升；每節氣/每輪迴自然衰退。

#### Scenario: 提升與衰減
- 首次抽取：clarity 至少提升至 40；再次重溫：+10（上限 100）。
- 每推進 1 節氣：-1（最低 0）；每走完 1 輪迴：-5。
- 佈局/花魂路線可影響提升/衰減係數（例如瀟湘館+黛玉路線減衰）。

#### Scenario: 顯示映射
- clarity 100–70：完整文段顯示；
- 69–40：縮句摘要；
- 39–10：關鍵詞；
- <10：模糊占位（僅標題或意象）。

### Requirement: 與淚水交互
特定淚水澆灌可喚醒相關記憶並直接提高 clarity。

#### Scenario: 葬花淚→葬花記憶
- 使用「葬花淚」時，若存在葬花線記憶，隨機一段 clarity +K（可疊加 buff）。
