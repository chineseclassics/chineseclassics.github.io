## ADDED Requirements

### Requirement: 視覺主風格與卡片式格子
介面採用淡彩中國風插畫＋扁平色塊；地圖以卡片式格子呈現小場景插畫＋文字標籤。

#### Scenario: 主要版型
- 頂部狀態列（輪迴/節氣/資源/AP）、左側地圖、右側資訊面板。
- 卡片含小插圖、標題、副標，hover/active 有陰影與放大。

### Requirement: 替換 Emoji→SVG 圖標
資源與系統圖示（靈石、絳珠、記憶、花魂、鳥靈）統一改為 SVG。

#### Scenario: 響應與暗色
- SVG 與文字均需在暗色/亮色模式清晰可辨。

### Requirement: 插畫策略（少而精）
每個主要院子 1–2 張插畫、主要花魂各 1 張插畫、大觀園遠景 1–2 張；其餘由 CSS 結構與色塊完成。

#### Scenario: AI 生成規範
- Prompt 關鍵詞：淡彩、水墨、小場景、簡化構圖、柔和色彩、留白。
- 素材以 PNG/WebP 嵌入卡片背景與詳情。
