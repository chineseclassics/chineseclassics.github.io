## ADDED Requirements

### Requirement: 回目路線選擇
每輪迴開始前，玩家在 2–3 個「回目路線卡」中選擇本輪主題，影響資源生成與事件權重。

#### Scenario: 路線卡示例
- 黛玉初入榮府線：黛玉早期記憶生成率↑，瀟湘館成本↓。
- 元妃省親線：富麗建築上限↑，維護成本↑。
- 抄檢大觀園線：衰敗加速，但維持關鍵建築可得額外結局分支。

#### Scenario: UI 與交互
- 節氣重置時彈出卡片式 UI，顯示路線簡述與圖示；選定後在頂部狀態列顯示「本輪追憶：XXX」。

### Requirement: 路線效果整合
路線選擇影響記憶生成候選、佈局 buff、生長倍率與事件機率，需與其他模組掛鉤。
