## ADDED Requirements

### Requirement: 花魂雙路線與品質標記
黛玉花魂根據近期使用淚水與情境傾向，形成不同培育路線與品質標記。

#### Scenario: 極致悲情路線
- 條件：最近 N 次澆灌中，葬花/焚稿/身世悲涼類淚水占比 ≥P%。
- 效果：單次共鳴值上限提高；觸發悲情相關事件機率上升；部分結局文本傾向悲絕。
- 視覺：花姿更冷豔，色調偏冷，粒子特效稀疏。

#### Scenario: 柔中帶剛路線
- 條件：最近 N 次澆灌中，姐妹情誼/詩社相聚/清幽日常類記憶占比 ≥P%。
- 效果：整體成長穩定；與佈局 buff 匹配時獎勵更均衡；結局文本偏向溫柔留白。
- 視覺：色調偏暖，粒子特效更柔和。

#### Scenario: 品質標記
- 根據路線與達成條件，標記 `quality = common | rare | sublime`，對應不同清晰度獲得上限或 UI 裝飾。

### Requirement: 路線可轉向與鎖定
允許在早期轉向；達到某級別或條件後進入「路線鎖定」。

#### Scenario: 轉向窗口
- 在黛玉花魂 Lv1–Lv2 可因連續的相反淚水搭配觸發轉向檢查。
- 進入 Lv3 後鎖定路線（可配置），僅影響細節加成，不再改變大方向。
