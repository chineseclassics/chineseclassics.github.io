# Core UI - Change Deltas

## ADDED Requirements

### Requirement: 房間列表界面
系統 SHALL 提供房間列表界面。

#### Scenario: 房間列表顯示
- 顯示「創建房間」按鈕
- 顯示「加入房間」按鈕（輸入房間碼）
- 顯示用戶認證狀態（登入/登出按鈕）
- 界面極簡設計，低調配色

#### Scenario: 房間列表布局
- 主要內容居中顯示
- 按鈕設計：圓角、細邊框、低調配色
- 輸入框設計：細邊框、低調焦點效果
- 整體布局簡潔，不堆積元素

### Requirement: 創建房間界面
系統 SHALL 提供創建房間界面。

#### Scenario: 創建房間表單
- 房間名稱輸入框
- 自定義詞語輸入區（多行文本，最多 600 字符）
- 遊戲設置（繪畫時間、輪數、提示數量）
- 創建按鈕

#### Scenario: 創建房間驗證
- 表單提交前驗證所有字段
- 驗證失敗時，顯示錯誤提示（低調設計）
- 驗證成功後，創建房間並跳轉到等待大廳

#### Scenario: 創建房間 UI
- 表單極簡設計（細線條邊框）
- 輸入框低調焦點效果
- 按鈕優雅的懸停效果
- 整體配色低調（灰色系）

### Requirement: 等待大廳界面
系統 SHALL 提供等待大廳界面。

#### Scenario: 等待大廳顯示
- 顯示房間信息（房間名稱、房間碼）
- 顯示玩家列表（所有已加入的玩家）
- 標記房主（特殊標識）
- 顯示房間設置（繪畫時間、輪數等）

#### Scenario: 等待大廳交互
- 房主可以點擊「開始遊戲」按鈕
- 其他玩家看到「等待房主開始」提示
- 玩家可以離開房間
- 玩家列表實時更新（新玩家加入時）

#### Scenario: 等待大廳 UI
- 玩家列表卡片設計（極簡）
- 開始按鈕低調設計
- 整體布局清晰，不堆積元素

### Requirement: 遊戲界面
系統 SHALL 提供遊戲界面。

#### Scenario: 遊戲界面布局
- 畫布區域：占據主要視覺空間（70-80%）
- 工具欄：位於畫布上方或側邊（不遮擋畫作）
- 猜詞區域：位於畫布下方或側邊（低調設計）
- 玩家信息區域：顯示排行榜、當前畫家、倒計時

#### Scenario: 畫布區域
- 畫布占據主要空間，白色背景
- 畫布邊框：極細的淺灰色邊框
- 畫布響應式設計（適配不同屏幕尺寸）
- 畫布支持高 DPI 顯示

#### Scenario: 工具欄
- 工具欄包含：顏色選擇、粗細調整、橡皮擦、清空
- 工具欄極簡設計（細線條圖標）
- 工具欄低調配色（不搶奪畫作焦點）
- 工具切換有輕微的動畫效果

#### Scenario: 猜詞區域
- 猜詞輸入框（支持中文輸入）
- 提示顯示區域（可選）
- 已猜中玩家列表
- 猜詞區域低調設計（細邊框、淺色背景）

#### Scenario: 玩家信息區域
- 分數排行榜（實時更新）
- 當前畫家標識
- 倒計時顯示
- 輪次信息（當前輪次 / 總輪次）

### Requirement: 極簡設計風格
系統 SHALL 應用極簡設計風格。

#### Scenario: 配色方案
- 背景：白色或極淺灰色（#FAFAFA）
- 文字：深灰色（#333333, #555555）
- 邊框：淺灰色（#E0E0E0, #CCCCCC）
- 強調：深灰色（#666666, #888888）

#### Scenario: 視覺元素
- 細線條邊框（1px）
- 極少或無陰影
- 圓角設計（4px）
- 低調的圖標（細線條）

#### Scenario: 動畫效果
- 適量的、優雅的、微妙的動畫
- 淡入淡出效果（頁面切換）
- 平滑的過渡（狀態變化）
- 輕微的懸停效果（按鈕、卡片）

### Requirement: 響應式設計
系統 SHALL 支持響應式設計。

#### Scenario: 桌面端布局
- 畫布和工具欄橫向布局
- 猜詞區域位於側邊或下方
- 玩家信息區域位於側邊

#### Scenario: 移動端布局
- 畫布占據主要空間
- 工具欄可收起/展開
- 猜詞區域位於下方
- 玩家信息區域可收起/展開

#### Scenario: 觸摸優化
- 工具按鈕足夠大（易於觸摸）
- 輸入框易於輸入
- 畫布支持觸摸繪畫

### Requirement: 錯誤提示
系統 SHALL 提供友好的錯誤提示。

#### Scenario: 錯誤提示顯示
- 錯誤提示低調設計（不搶奪畫作焦點）
- 錯誤提示位置：不遮擋主要內容
- 錯誤提示自動消失（3-5 秒後）
- 錯誤提示可以手動關閉

#### Scenario: 錯誤類型
- 網絡錯誤：顯示「網絡連接失敗，請重試」
- 權限錯誤：顯示「您沒有權限執行此操作」
- 驗證錯誤：顯示具體的驗證失敗原因
- 其他錯誤：顯示通用錯誤提示
