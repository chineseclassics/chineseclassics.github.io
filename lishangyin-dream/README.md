# 碧城・煉夢錄

一款基於李商隱詩歌意象的合成遊戲，讓學生在遊戲中理解詩歌的意象組合與情感表達。

## 遊戲簡介

玩家扮演「碧城掌燈人」，在一個名為「碧城」的記憶圖書館中，通過組合散落的「意象碎片」，修復「無題詩人」破碎的靈魂記憶。

## 核心玩法

1. **拾遺囊**：收集散落的意象碎片（如：春蠶、蠟炬、蝴蝶、藍田等）
2. **靈犀爐**：將兩個意象拖入靈犀爐進行煉化
3. **夢境重構**：成功組合後，觸發「記憶閃回」，解鎖完整的詩句和敘事
4. **無題卷**：查看已修復的記憶和意象解析

## 教學目標

- 理解李商隱詩歌中的核心意象
- 掌握意象之間的邏輯聯繫
- 體驗詩歌的情感表達與意境
- 培養對古典詩歌的興趣

## 技術實現

- 純 HTML/CSS/JavaScript
- 響應式設計，支持桌面和移動端
- 拖拽合成機制（支持觸摸操作）

## 使用方式

直接在瀏覽器中打開 `index.html` 即可開始遊戲。

## 開發說明

- `js/data.js`：詩歌數據與配方系統
- `js/game.js`：遊戲核心邏輯
- `css/style.css`：碧城主題樣式
