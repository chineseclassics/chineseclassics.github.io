# 隊伍徽章圖標

此目錄存放隊伍使用的豆製品圓形圖標。

## 所有豆製品圖標

目前系統支持 8 種豆製品：

- `豆芽.png` - 生機勃勃
- `豆乾.png` - 堅實耐嚼
- `豆腐.png` - 柔軟細膩
- `豆包.png` - 圓潤飽滿
- `豆豉.png` - 發酵深度
- `豆漿.png` - 流暢滋養
- `油豆腐.png` - 外酥內嫩
- `豆苗.png` - 清新翠綠

## 規格要求

- 格式：PNG（支持透明背景）
- 形狀：圓形
- 尺寸：建議 256x256 或更高解析度
- 風格：卡通風格，與用戶頭像系統風格一致

## 使用

這些圖標將用於：
1. **課堂鬥豆的隊伍標識**（目前使用前 4 個：豆芽、豆乾、豆腐、豆包）
2. **未來的農場成就徽章系統**（共用同一套圖標，可擴展使用所有 8 個）
